# StudentRegistryServletApp

## Overview
Java Servlet-based web application for student registration and retrieval using JDBC & MySQL.

## Features
- Register students
- View all registered students
- Persistent storage with MySQL

## Technologies
- Java Servlets
- JDBC
- MySQL
- Apache Tomcat

## Run Instructions
1. Create MySQL database using `database/schema.sql`
2. Add MySQL JDBC driver to `WebContent/WEB-INF/lib`
3. Deploy on Apache Tomcat
4. Open `index.html` to register students
5. Visit `/show_all` to view all students
